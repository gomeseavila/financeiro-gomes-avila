# Gomes & Ávila | Financeiro Empresarial

Projeto React + Vite pronto para publicar no Vercel.

## Como rodar localmente

```bash
npm install
npm run dev
```

## Como publicar no Vercel

1. Suba esta pasta para um repositório no GitHub.
2. Entre em https://vercel.com
3. Clique em **Add New > Project**.
4. Escolha este repositório.
5. Clique em **Deploy**.

## Funcionalidades incluídas

- Dashboard financeiro
- Cadastro de clientes
- Cadastro de empresas
- Plano de contas editável
- Centros de custos
- Lançamentos manuais
- Importação de extratos XLSX/CSV
- Conciliação bancária simples
- DRE resumida
- DRE mensal automática
- Fluxo de caixa
- Exportação Excel e PDF
- Layout responsivo

## Observação

Esta versão salva os dados no navegador usando localStorage.
Para login real, banco de dados em nuvem e multiusuário real, a próxima etapa é integrar Supabase.
